# AI Agent Skills

Files in this directory are meant to be used by AI agents working with
PostGIS.

Skills are a standardized way to give AI agents project-specific capability
and expertise; see <https://agentskills.io/> for the format.

PostGIS keeps these files in this repository so agent guidance is reviewed
with the code and documentation it describes, follows the same branch history,
and can point directly at maintained developer and user documentation. Keep
skills advisory: they should steer agents toward current project practice, but
they do not replace the manual, developer docs, tests, tickets, mailing-list
decisions, or maintainer review.

When a skill needs long-form background, put that background in the maintained
documentation and link to it from the skill. This keeps the skills concise and
keeps project policy in the same documentation surfaces human contributors use.
